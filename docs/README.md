# Documentation

This plugin provides tools for code highlighting and markup conversion.

## Highlighter

Code syntax highlighting for displaying source code snippets.

See [Highlighter.md](Highlighter.md) for details.

## Formatters

Markup-to-HTML converters for various formats:

- **[Markdown](Markdown.md)** - CommonMark/GFM support via league/commonmark
- **[BBCode](Bbcode.md)** - Forum-style markup via mjohnson/decoda
- **[Djot](Djot.md)** - Modern markup language via php-collective/djot
